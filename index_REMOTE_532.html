Comandos Git

Configura o autor do git

git config --global user.name "{Nome do usúario}"

O comando configura o nome para anexar os commits

git config --global user.mail "{email do usúario}"

O comando configura o email para anexar aos commits.

Cria um repositório Git

git init

O comando, transforma uma pasta em repositório Git

Adiciona arquivo a um repositório

git add {nome do arquivo}

O comando adiciona um arquivo ou vários arquivos ou estágio de commit, para adicinar vários basta separar o nome com espaço ou usar o "." no lugar do nome.

Confirma a alteração do arquivo

git commit "{Mensagem de commit}"

O comando confirma as alterações com uma mensangem que deve ser descritiva e e direta

Altera a visualização do repositório

git ckeckout {id do commit || nome da branch}

Cria uma ramificação do repositório

git checkout -b {nome-da-branch}

O comando cria uma nova ramificação do repositório

Resolve um confito de merge

git margetool

O comando abre um editor para resolver conflitos de merge
